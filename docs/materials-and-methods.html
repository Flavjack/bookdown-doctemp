2 Materials and Methods

The data was analyzed in the statistical software R (R Core Team, 2018). The germination analysis and graphics was carried out with the package GerminaR (Lozano Isla, Benites Alfaro, & Pompelli, 2017). Each variable was submitted at analysis of variance (ANOVA) and the mean comparison test used was Student-Newman Keuls (P<0.05)(de Mendiburu, 2017). For the multivariate analysis, the principal components analysis (PCA) and cluster hierarchical classification analysis (HCPC) will be used (Husson, Josse, Le, & Mazet, 2018). The report was generated with the FactoInvestigate package (Thuleau & Husson, 2018).
